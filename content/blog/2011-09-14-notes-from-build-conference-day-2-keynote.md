Here are my raw, unedited notes from the Build conference day 2 keynote. The theme was Windows Server 8, Azure, and Visual Studio 2011.

## Highlights

- A multiplayer game demonstration showed Windows Phone 7 and Windows 8 communicating through Azure
- Productivity Power Tools reached over 1 million downloads and will be installed by default in VS2011
- Microsoft projected over 350 million installations of Windows shipping that year

## Visual Studio 2011

- Find matching code clones functionality
- Integrated image editor supporting 3D model loading
- A DOM explorer, kind of like Firebug inside of VS

## ASP.NET and MVC4

- New HTML5-based template with mobile optimization
- New async keyword used for controllers
- jQuery Mobile shipping by default
- Bundling and minification capabilities for CSS and JavaScript

## .NET 4.5 and Infrastructure

- WebSocketHandler introduced in .NET 4.5
- TFS 2011 hosted as Team Foundation Service in Azure
- Windows Server 8 introduced Storage Spaces technology for virtual machine capabilities

## Azure and Cloud

- Azure data platform expansion including SQL Azure, Table Storage, and Access Control Service
- Access Control Service to reduce development overhead for authentication

## Windows Phone

- Windows Phone 7.5 reached 30,000 apps
